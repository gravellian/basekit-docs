Accent (H2)

Base
tint 12%
tint 18%
shade 12%
shade 18%
Good defaults: hover → tint 12–18%, active → shade 12–18%.
// SCSS equivalents
$accent-base:  $brand-accent;
$accent-t12:   tint($brand-accent, 12%);
$accent-t18:   tint($brand-accent, 18%);
$accent-s12:   shade($brand-accent, 12%);
$accent-s18:   shade($brand-accent, 18%);

// Example usage
a:hover  { color: $accent-t12; }
a:active { color: $accent-s18; }
.h2-underline { border-bottom: $stroke-2 solid rgba($accent-t12, .85); }
